# Setup Runbook

## Purpose
Initialize local development and verify baseline project health.

## Steps
1. Ensure Python 3 is installed (`python --version`).
2. From repo root, run: `python -m http.server 5500`.
3. Open `http://127.0.0.1:5500/web/`.
4. Confirm the starter dashboard renders and updates status text.

## Success Criteria
- Local server starts without errors.
- Web UI loads and interactive button works.
